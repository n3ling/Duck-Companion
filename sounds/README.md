# Custom duck sounds (optional)

Place your own audio files here to replace the built-in synthesized sounds. If a file is missing, the extension falls back to the default sound.

| File       | Used for              |
|-----------|------------------------|
| `quack.mp3` | Click-to-quack        |
| `happy.mp3` | Pet mode (happy chirp) |
| `eat.mp3`   | Feed mode (eating)    |

**Format:** MP3 is recommended. Short clips (under 1 second) work best.

**Naming:** Use the exact filenames above. Other files in this folder are ignored for playback.
